# Foundations Brand Spec

## Colors

| Name | Hex | Usage |
|---|---|---|
| Ion Violet | `#7C56FE` | Primary brand color — CTAs, accents, key emphasis |
| Ignite | `#FF582A` | Secondary — use sparingly for attention/alert states |
| Arctic Steel | `#A2C1C4` | Secondary — muted, supporting elements |
| Ice Blue | `#CFE7FF` | Tertiary — backgrounds, subtle accents |
| Azure | `#B6BBFF` | Tertiary |
| Pulse | `#57FF60` | Tertiary — success / positive states |
| Obsidian Black | `#090D19` | **Use instead of true black** — primary dark |
| Alloy Silver | `#EFEFEF` | **Use instead of true white** — primary light |

Critical: never use `#000000` or `#FFFFFF`. Always use Obsidian Black and Alloy Silver.

## Typography

- **Primary typeface:** PP Valve (licensed commercial font — not available publicly)
- **Fallback for web:** Inter (Google Fonts) — closest open-source grotesque
- **Alternative fallback:** IBM Plex Sans
- **System fallback:** Helvetica, Arial, sans-serif
- **Weights used:** 300 (Light), 400 (Regular), 500 (Medium), 700 (Bold)
- Use Inter throughout the Streamlit app. Load via Google Fonts.

## Typography rules

- Left-aligned, rag right. No centered or justified multi-line text.
- No strokes, outlines, or drop shadows on type.
- No rotation other than 0° or 90°.
- Use contrast between weights and sizes to create hierarchy — when pairing two weights, skip a weight (e.g., Regular + Bold, not Regular + Medium). When pairing two sizes, double the size.

## Logo rules

- Do not alter, recolor, skew, stretch, or add effects to the logo.
- Clear space = height of the icon on all four sides.
- Never on busy or high-contrast backgrounds.
- Primary lockup is the default. Use it in the app header.

## Design philosophy

- Expansive negative space.
- High contrast for legibility.
- Minimal, confident, tech-forward — this is a B2B tech brand.
- The violet is the signature — use it deliberately, not decoratively.
