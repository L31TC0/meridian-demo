# Brand Assets

This folder contains brand assets and guidelines for the Foundations demo apps.

## Contents

- `foundations_logo_primary_reversed.png` — Primary horizontal lockup (violet icon + off-white wordmark on Obsidian Black). Dropped in manually.
- `brand_spec.md` — Full brand specification: colors, typography, logo rules, design philosophy.

## Usage

`brand_spec.md` is the **authoritative reference** when styling any part of the demo app. All color values, type choices, and design rules should be sourced from that file.

PP Valve is Foundations' primary typeface but is a licensed commercial font not available publicly. **Use Inter (Google Fonts) as the web substitute** throughout all Streamlit apps and web-based demos.
